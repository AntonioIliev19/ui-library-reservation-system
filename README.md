## Storybook

I use [Storybook](https://storybook.js.org/) to develop, showcase, and test my components in isolation.

## Installation

To install this library in your Angular project, run:

```bash
npm install reservation-library
```

## To Run Storybook

Please use ng run {{projectName}}:storybook.
For example: ng run ui:storybook.
